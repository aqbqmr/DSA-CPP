# DSA C++

DSA and problem-solving in C++ with a cleaner setup for daily question practice.

## Recommended layout

- `problems/leetcode/leetcode_ques_no_1222.cpp`: solve one LeetCode problem per file
- `inputs/leetcode/leetcode_ques_no_1222.txt`: optional custom input for local testing
- `build/`: compiled files generated by VS Code tasks and scripts
- `scripts/run_cpp_file.sh`: compile or run the current `.cpp` file
- `scripts/new_leetcode_problem.sh`: create a new LeetCode file quickly

## Daily workflow

1. Create a new file with a question number:
   `./scripts/new_leetcode_problem.sh 1222`
2. Write your solution in:
   `problems/leetcode/leetcode_ques_no_1222.cpp`
3. Add optional input in:
   `inputs/leetcode/leetcode_ques_no_1222.txt`
4. In VS Code, open that `.cpp` file and run the default build task.
5. The compiled binary goes into `build/` instead of beside your source file.

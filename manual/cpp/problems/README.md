# Problems

Use this folder for handwritten problem practice.

Examples:

- `manual/cpp/problems/binary_search.cpp`
- `manual/cpp/problems/valid_parentheses.cpp`
- `manual/cpp/problems/two_sum.cpp`

Optional:

- keep a matching `.md` file beside each `.cpp` file for notes
- keep an `input.txt` or `same_name.txt` file beside a source file if you want local input
